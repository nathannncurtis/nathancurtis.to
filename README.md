# nathancurtis.to

Personal portfolio site. Static HTML/CSS/JS. Hosted on GitHub Pages.

## Local Development

Open `index.html` in a browser, or serve locally:

```bash
python -m http.server 8000
```

## Deployment

Push to `main`. Configure GitHub Pages to serve from the root of `main`.

For custom domain, add a `CNAME` file containing `nathancurtis.to`.
